### Installation Guide :

```
pip install -r requirements.txt
```


create a .env file in  your main folder  
and Add 

Get your Google Gemini Api Key at: https://aistudio.google.com/app/apikey

```
GEMINI_API_KEY = "YOUR_GOOGLE_GEMINI_API_KEY"
```


Run the Code 

```
python gradio_ui.py
```
